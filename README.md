
This repo proposes updated ACML performance scripts to compare with atlas, openblas and netlib.

# try it

get the ACML distribution file and run:
```
./unarchive-here <path-to-archive.tgz>
```

then

```
cd gfortran64/examples/perfcomp
./makeme
```
